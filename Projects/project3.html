Game(Still in development)

A mining-style simulation game focused on exploration and progression.

Overview

This simulation game was built with the goal of blending exploration, collection, and progression. Players dig through themed maps to uncover and collect fossils, unlocking new tools, pets, and areas along the way. It features multiple biomes, a tool-based upgrade system, and mechanics like selling, hatching pets, and auto-saving progress. The game was designed for both casual fun and long-term engagement.

Details

  • Platform: Roblox
  • Tech Stack: Roblox Studio (Lua), UI design, client-server architecture
  • Key Features: Progressive digging, unlockable tools, egg hatch system, pet display, area unlocking
