Green Block Game

A basic sequence memory game.

Overview

GreenBlockGame was my very first mini project — a simple yet engaging memory challenge built to test focus and short-term recall. The game features a 6x6 grid of numbered blocks. It begins by showing one number in a sequence. Each time you click the correct block, a new number is added to the sequence, making it progressively harder. If you miss even one, the game ends. It’s a straightforward but effective way to sharpen your memory!

  • Tech Stack: HTML, CSS, JS.
