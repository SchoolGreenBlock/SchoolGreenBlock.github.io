MonuScan

Scan a monument, get instant info. Simple and smart.

Overview

MonuScan is a mobile application that identifies historical monuments using a device's camera. When the user points the camera at a monument, the app recognizes it and provides detailed information — including history, construction details, and cultural significance. This project focused on blending machine learning with real-time scanning to create an interactive educational tool.

Details

  • Tech Stack: Dart, Flutter, Python, TensorFlow Lite, Flask
  • Key Features:
    • Camera-based monument recognition
    • Offline-capable TFLite model
    • Instant display of monument info
